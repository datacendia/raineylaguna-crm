# 01 — Getting Started

**Who this is for:** anyone who logs into the CRM.
**What you'll do:** sign in safely, turn on two-factor security, and learn what
each page of the dashboard is for.

---

## Signing in

1. Go to the CRM web address (`crm.raineylaguna.com`).
2. Enter your **email** and **password**.
3. If two-factor security is switched on for your account, you'll then be asked
   for a **6-digit code**. Open your authenticator app (Google Authenticator,
   Authy, etc.), read the current code for "Rainey Laguna", and type it in.
4. You're in — you'll land on the dashboard.

> 📷 Screenshot: the login screen.

> ⚠️ If something looks wrong: after 5 wrong tries in a minute the system makes
> you wait a minute before trying again (this protects the account). Wait, then
> try again. Lost your password or your authenticator? See
> [Troubleshooting](06-troubleshooting.md).

## Two-factor security (recommended)

Two-factor (also called "2FA" or "TOTP") means that even if someone learns your
password, they still can't get in without the code on your phone.

1. Open the **Security** page from the left menu.
2. Follow the steps to **set up two-factor**: scan the QR code with an
   authenticator app, then enter a code once to confirm it works.
3. From then on, you'll enter a fresh code each time you sign in.

> 📷 Screenshot: the Security page with the 2FA setup.

## Staying signed in

You stay signed in while you're active. If you don't use the CRM for **7 days**
you'll be signed out, and after **30 days** you always sign in fresh. This is
normal and keeps the account safe.

## A tour of the dashboard

The left menu is your map. Here's what each page is for:

| Page | What it's for |
|------|---------------|
| **Leads** | The master list of every business (prospect). Search, filter, and open any one to see its detail. This is where you spend most of your time. |
| **Pipeline** | The same leads, arranged by stage (Lead → Contacted → Audited → Proposal → Closed) so you can see where everyone is at a glance. |
| **Drafts** | Messages the system has written for you, waiting for your review before they go out. |
| **Outreach** | A log of messages that have been sent and what happened (delivered, opened, replied). |
| **Digest** | The weekly summary — new leads, who's gone quiet, wins. The same numbers also arrive by email every Monday. |
| **Video audits** | Where you track short video walkthroughs (Looms) prepared for a lead. |
| **Batch** | Tools for working on many leads at once. |
| **Security** | Your account safety — password and two-factor. |
| **Guide** | Quick in-app reference. |
