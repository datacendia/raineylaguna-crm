# 02 — Daily & Weekly Workflow

**Who this is for:** the person running the CRM each day.
**What you'll do:** turn new prospects into contacted, audited, and (eventually)
won clients — using the same simple loop every day.

---

## The daily loop

Do this once a day. It usually takes 20–40 minutes.

1. **Open the Leads page.** Sort or filter to find **new** leads (stage = "Lead")
   you haven't contacted yet. Tip: filter by **Source** to focus (e.g. people who
   came from the website audit are warmer).
2. **Open a lead** by clicking its name. You'll see its details: district, niche,
   website, and — if we have one — its **Digital Audit** (a 0–100 score and a list
   of concrete problems with their site).
3. **Read the audit.** This is your ammunition. The problems listed (e.g. "no own
   website", "site not secure", "slow on mobile") are what your message will open
   with. If a prospect ran the audit themselves on our site, you can open the exact
   report they saw.
4. **Generate a draft message.** Use the lead's **draft / generate outreach**
   action. The system writes a first message for you — automatically in the right
   **language** (Spanish for Peru/Latin America, English for the US/UK) and on the
   right **channel** (WhatsApp for Peru, Email elsewhere).
5. **Review and edit it.** Read the draft on the **Drafts** page. Make sure it
   sounds right and every fact is true. Fix anything. **You are the editor — never
   send a draft you haven't read.**
6. **Send it.** Use the **Send** action on the draft. The system delivers it and
   records it under **Outreach**.
7. **Log what happens.** When the prospect replies, update the lead: move its
   **stage** forward (e.g. to "Contacted"), and set a **next action** (e.g. "send
   proposal Friday") so it doesn't fall through the cracks.

### Daily checklist

- [ ] Reviewed new leads
- [ ] Generated + **edited** + sent drafts (read every one)
- [ ] Logged replies and set a next action on each touched lead

## The weekly loop (Mondays)

1. **Read the Monday digest** (the Digest page, or the email that arrives Monday
   morning). It shows: leads added this week, outreach sent, proposals out, wins,
   and importantly **who's gone cold**.
2. **Chase the "going cold" list** — leads contacted 14+ days ago with no reply.
   Send a short, friendly follow-up.
3. **Glance at the Pipeline** to make sure nobody important is stuck.

### Weekly checklist

- [ ] Read the Monday digest
- [ ] Followed up everyone going cold (14+ days)
- [ ] Pipeline reviewed

## A few golden rules

- **Read before you send.** Drafts are a starting point, not a finished message.
- **Only true facts.** If the message says something about their business, it must
  be real (from the audit or their listing). The system is built to avoid making
  things up — keep it that way when you edit.
- **One message, then wait.** Don't pile on. A reply is the goal, not volume.
