# 01 — Primeros pasos

**Para quién es:** cualquiera que ingrese al CRM.
**Qué vas a hacer:** ingresar de forma segura, activar la seguridad de dos pasos y
conocer para qué sirve cada página del tablero.

---

## Ingresar

1. Entra a la dirección del CRM (`crm.raineylaguna.com`).
2. Escribe tu **correo** y tu **contraseña**.
3. Si tu cuenta tiene activada la seguridad de dos pasos, te pedirá un **código de
   6 dígitos**. Abre tu app de autenticación (Google Authenticator, Authy, etc.),
   busca el código actual de "Rainey Laguna" y escríbelo.
4. Listo — entrarás al tablero.

> 📷 Captura: la pantalla de ingreso.

> ⚠️ Si algo se ve raro: después de 5 intentos fallidos en un minuto, el sistema te
> hace esperar un minuto (esto protege la cuenta). Espera y vuelve a intentar.
> ¿Perdiste la contraseña o tu app de autenticación? Ve a
> [Solución de problemas](06-solucion-de-problemas.md).

## Seguridad de dos pasos (recomendado)

La seguridad de dos pasos (también llamada "2FA" o "TOTP") significa que, aunque
alguien sepa tu contraseña, igual no puede entrar sin el código de tu teléfono.

1. Abre la página **Security** (Seguridad) en el menú de la izquierda.
2. Sigue los pasos para **activar los dos pasos**: escanea el código QR con una app
   de autenticación y luego escribe un código una vez para confirmar que funciona.
3. Desde ahí, escribirás un código nuevo cada vez que ingreses.

> 📷 Captura: la página Security con la configuración de 2FA.

## Mantenerte conectada

Te mantienes conectada mientras estés activa. Si no usas el CRM por **7 días** te
cerrará la sesión, y después de **30 días** siempre vuelves a ingresar desde cero.
Es normal y mantiene la cuenta segura.

## Recorrido por el tablero

El menú de la izquierda es tu mapa. Esto es para qué sirve cada página:

| Página | Para qué sirve |
|--------|----------------|
| **Leads** | La lista maestra de cada negocio (prospecto). Busca, filtra y abre cualquiera para ver su detalle. Aquí pasas la mayor parte del tiempo. |
| **Pipeline** | Los mismos leads, ordenados por etapa (Lead → Contacted → Audited → Proposal → Closed) para ver de un vistazo dónde está cada uno. |
| **Drafts** | Mensajes que el sistema escribió por ti, esperando tu revisión antes de salir. |
| **Outreach** | El registro de los mensajes enviados y qué pasó (entregado, abierto, respondido). |
| **Digest** | El resumen semanal — nuevos leads, quién se enfrió, cierres. Los mismos números llegan por correo cada lunes. |
| **Video audits** | Donde llevas el control de videos cortos (Looms) preparados para un lead. |
| **Batch** | Herramientas para trabajar varios leads a la vez. |
| **Security** | La seguridad de tu cuenta — contraseña y dos pasos. |
| **Guide** | Una guía rápida dentro de la app. |

> 📷 Captura: el tablero con el menú de la izquierda señalado.
