# Rainey Laguna — Operations Handbook

This is the day-to-day operating manual for the Rainey Laguna system: the **CRM**
(`raineylaguna-crm`, where you manage and contact prospects) and the **marketing
site** (`raineylaguna.com`, where prospects find you and run a free audit).

Written for the person running the CRM every day. You do **not** need to be
technical. Setup and server topics are marked "for the founder".

## How to use this handbook

- New here? Read **00 — Overview**, then **01 — Getting Started**, then **02 — Daily Workflow**.
- Not sure how you're allowed to contact someone? **05 — Compliance & Safety** is the rulebook.
- Something not working? **06 — Troubleshooting**.
