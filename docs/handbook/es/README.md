# Rainey Laguna — Manual de Operaciones

Este es el manual de uso diario del sistema Rainey Laguna: el **CRM**
(`raineylaguna-crm`, donde gestionas y contactas prospectos) y la **página web**
(`raineylaguna.com`, donde los prospectos te encuentran y corren una auditoría gratis).

Está escrito para la persona que usa el CRM todos los días. **No** necesitas ser
técnica. Los temas de configuración o de servidores están marcados como "para el fundador".

> Nota: dentro del CRM, los nombres de las páginas y botones están en inglés
> (Leads, Drafts, Outreach…). En este manual los dejamos en inglés tal como los ves
> en pantalla, y te explicamos todo en español.

## Cómo usar este manual

- ¿Recién empiezas? Lee **01 — Primeros pasos**, luego **02 — Flujo diario**.
- ¿No sabes si puedes contactar a alguien de cierta forma? **05 — Cumplimiento y seguridad** es el reglamento.
- ¿Algo no funciona? **06 — Solución de problemas**.
